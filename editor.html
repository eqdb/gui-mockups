1. Position vector
\vec{p} $$=$$ p_x\cdot\hat{i}+p_y\cdot\hat{j} make rule
$$=$$ r\left(\cos\theta\cdot\hat{i}+\sin\theta\cdot\hat{j}\right) (rule) remove rule
+

2. Find velocity vector
\vec{v} $$=$$ \frac{\partial}{\partial t}\vec{p} make rule
$$=$$ \frac{\partial}{\partial t}\left( r\left(\cos\theta\cdot\hat{i}+\sin\theta\cdot\hat{j}\right)\right) make rule
$$=$$ r\frac{\partial\theta}{\partial t} \left(-\sin\theta\cdot\hat{i}+\cos\theta\cdot\hat{j}\right) (rule) remove rule
+

3. Find acceleration vector
\vec{a} $$=$$ \frac{\partial}{\partial t}\vec{v} make rule
$$=$$ \frac{\partial}{\partial t}\left( r\frac{\partial\theta}{\partial t} \left(-\sin\theta\cdot\hat{i}+\cos\theta\cdot\hat{j}\right) \right) make rule
$$=$$ -r\left(\frac{\partial\theta}{\partial t}\right)^2 \left(\cos\theta\cdot\hat{i}+\sin\theta\cdot\hat{j}\right) (rule) remove rule
+

4. Find acceleration magnitude
a $$=$$ \left|\vec{a}\right| make rule
$$=$$ \left|-r\left(\frac{\partial\theta}{\partial t}\right)^2 \left(\cos\theta\cdot\hat{i}+\sin\theta\cdot\hat{j}\right)\right| make rule
$$=$$ r\left(\frac{\partial\theta}{\partial t}\right)^2 make rule
$$=$$ \frac{v^2}{r} (rule) remove rule
+
