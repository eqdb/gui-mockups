Centripetal Acceleration

Steps

  1. Position vector
  2. Find velocity vector
  3. Find acceleration vector
  4. Find acceleration magnitude

1. Position vector
$$\vec{p}$$ $$=$$ $$\left(\begin{matrix}p_x\\p_y\end{matrix}\right)$$
  • Substitute $$p_x=r\cos\theta$$
  • Substitute $$p_y=r\sin\theta$$
$$=$$ $$r\left(\begin{matrix}\cos\theta\\\sin\theta\end{matrix}\right)$$

2. Find velocity vector
$$\vec{v}$$ $$=$$ $$\frac{\partial}{\partial t}\vec{p}$$
  • Substitute $$\vec{p}$$
$$=$$ $$\frac{\partial}{\partial t}\left( r\left(\begin{matrix}\cos\theta\\\sin\theta\end{matrix}\right) \right)$$
  • Simplify (calculus/algebra)
$$=$$ $$r\frac{\partial\theta}{\partial t} \left(\begin{matrix} -\sin\theta\\\cos\theta \end{matrix}\right)$$

3. Find acceleration vector
$$\vec{a}$$ $$=$$ $$\frac{\partial}{\partial t}\vec{v}$$
  • Substitute $$\vec{v}$$
$$=$$ $$\frac{\partial}{\partial t}\left( r\frac{\partial\theta}{\partial t} \left(\begin{matrix} -\sin\theta\\\cos\theta \end{matrix}\right) \right)$$
  • Simplify (calculus/algebra)
$$=$$ $$-r\left(\frac{\partial\theta}{\partial t}\right)^2 \left(\begin{matrix} \cos\theta\\\sin\theta \end{matrix}\right)$$

4. Find acceleration magnitude
$$a$$ $$=$$ $$\left\lVert\enspace\vec{a}\enspace\right\rVert$$
  • Substitute $$\vec{a}$$
$$=$$ $$\left\lVert\enspace -r\left(\frac{\partial\theta}{\partial t}\right)^2 \left(\begin{matrix} \cos\theta\\\sin\theta \end{matrix}\right)\enspace\right\rVert$$
  • Simplify (algebra/trigonometry)
$$=$$ $$r\left(\frac{\partial\theta}{\partial t}\right)^2$$
  • Substitute $$\frac{\partial\theta}{\partial t}=\frac{v}{r}$$
$$=$$ $$\frac{v^2}{r}$$
